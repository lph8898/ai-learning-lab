# AI Chatbot (Starter Project)

## Overview

This project is a simple console-based chatbot used as a starting point in the AI Learning Lab.
It is intentionally minimal and rule-based so that it can be gradually upgraded with real AI models.

## Features

- Runs in a terminal or command prompt
- Simple rule-based responses
- Clear extension point for integrating AI APIs (e.g., OpenAI, Anthropic, etc.)
